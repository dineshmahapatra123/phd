# Codex Workflow: ingest-paper

Purpose: find and download an open-access paper into the reading queue.

## Steps

1. Search for the requested paper and identify an open, legal PDF URL.
2. Prefer official publisher, author, institutional, SSRN, NBER, or arXiv sources.
3. Save into `8 - Doc_dump/Automated_Search/` with a clean title.
4. Do not move it into `7 - Raw/` unless the user asks.
5. Report the saved file path and suggested next step.
