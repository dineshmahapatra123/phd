# Codex Style Policy

This file resolves style-rule conflicts for Codex without modifying the existing Claude or Antigravity layers.

## Authority Order

When writing or checking PhD prose, use this order:

1. `9 - Knowledge_base/TISS_STYLE.md`
2. `9 - Knowledge_base/PHD_SCHEMA.md`
3. `Types/`
4. `CLAUDE.md`
5. `.agent/memory/tiss_style.md`

If any file conflicts with `TISS_STYLE.md`, follow `TISS_STYLE.md`.

## Citation Rule

Use APA author-date citation formatting as specified in `TISS_STYLE.md` section 9:

- Single author: `(Sharma, 2005)`
- Direct quote: `(Sharma, 2005, p. 34)`
- Page range: `(Sharma, 2005, pp. 34–36)`
- Two authors in parentheses: `(Gupta & Singh, 2001)`
- Two authors in prose: `Gupta and Singh (2001)`

Do not use the no-comma or colon-page form for chapter prose unless the user explicitly asks for a different style.
