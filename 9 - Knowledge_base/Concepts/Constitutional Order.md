## Standard Definition
Fundamental rules about how society is organized—the "rules for making rules." It provides the overarching framework within which institutional arrangements are created and modified.

## Scholarly Debate
Feder and Feeny (1991) differentiate Constitutional Order from [[Institutional Arrangements]] and [[Normative Behavioral Codes]], noting that it evolves slowly compared to specific arrangements.

## PhD Application
Useful for analyzing how top-level political changes (like post-colonial transitions in India) create the boundary conditions for land reform and property rights shifts.

## Measurement / Methods
Qualitative analysis of constitutional law, foundational legal principles, and state architecture.
