# Secure Property Rights

Seed note for Secure Property Rights extracted from Pradeep Nayak's book.

## Expanded Definition (Nayak, 2020)
The neoliberal ideal wherein property ownership is formally documented, legally protected, and easily transferable. It is seen as a prerequisite for functioning markets, as it allows assets to be used as collateral (capitalizing 'dead capital') and incentivizes long-term investment.
